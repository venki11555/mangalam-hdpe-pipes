# Run and deploy your React app (CRA)

This project now uses Create React App (Webpack) instead of Vite.

## Run Locally

**Prerequisites:** Node.js

1) Install dependencies: `npm install`
2) Start the app: `npm start`
3) Build for production: `npm run build`
4) (Optional) Set `GEMINI_API_KEY` in `.env` if you add Gemini features later.
